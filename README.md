# RDR Freeparty

Site web communautaire de réduction des risques pour soirée, teuf et freeparty.

## Installation GitHub Pages

1. Crée un dépôt GitHub.
2. Clique sur `Add file` puis `Upload files`.
3. Glisse **le contenu de ce dossier** à la racine du dépôt.
4. Clique sur `Commit changes`.
5. Va dans `Settings` → `Pages`.
6. Choisis :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/root`
7. Clique sur `Save`.

Le site sera disponible à une adresse du type :

`https://tonpseudo.github.io/nom-du-repo/`

## Important

Le fichier `index.html` doit être directement à la racine du dépôt GitHub.
Ne mets pas le dossier complet dans GitHub : ouvre le dossier, fais `Cmd + A`, puis glisse les fichiers.
