# Sources et attribution

Ce projet reprend une logique d’information RDR communautaire à partir des flyers fournis.

Les PDF originaux sont placés dans `pdf-originaux/`.

Les documents originaux doivent être conservés intacts, cités, et utilisés selon leur licence respective.

Images :
- `interactions-tripsit-guide.png` : tableau visuel TripSit des combinaisons.
- `benzos-etoiles-prevention.png` : support visuel prévention benzodiazépines fourni par l’utilisateur.

Cette app ne remplace pas les secours, un avis médical ou une équipe RDR.
