# Installation rapide

## Ce qu’il faut envoyer sur GitHub

À la racine du dépôt, tu dois avoir :

```text
index.html
interactions-tripsit-guide.png
benzos-etoiles-prevention.png
pdf-originaux/
README.md
NOTICE_SOURCES.md
INSTALLATION.md
.nojekyll
```

Si tu vois un dossier `UPLOAD_GITHUB_RDR_FREEPARTY` dans GitHub, ce n’est pas bon : il faut envoyer le contenu du dossier, pas le dossier lui-même.
